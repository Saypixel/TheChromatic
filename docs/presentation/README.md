### 필요한 패키지 설치 및 게임 실행
![install](./shell_1.png)
![run](./shell_2.png)\
`pip install -r requirements.txt` 명령어를 쉘에 입력하면 The Chromatic의 필요한 패키지를 설치됩니다.\
`python main.py` 명령어를 쉘에 입력하면 게임이 실행 됩니다.
