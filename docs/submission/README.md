# 게임 흐름
훈련장에 가는 문을 열고, 훈련장에서 장애물을 피하고 적을 쓰러뜨리며 미니 퍼즐을 푸는 게임

# 게임 방법
- `A`, `D` / `←`, `→`: 이동
- `W` / `↑` / `Space`: 점프 / NPC와 대화
- `ESC`: 일시중지 화면 표시 (인게임 내) / 게임 종료
- `J`: 기본 공격
- `E`: 아이템 사용
- `Shift`: 아이템 변경
- `R`: 시간을 5초 전으로 되돌리기
